Mode II Crack Propagation

Figure 1: boundary conditions of the Mode-II problem

The Mode-II problem is defined in the same fashion as the Mode-I problem. Boundary conditions, as described in Figure 1, are defined in the mechanical MultiApp by:

[BCs]
  [./xdisp]
    type = ScalarDirichletBC
    variable = 'disp_x'
    boundary = 'top'
    scalar_var = 'load'
  [../]
  [./yfix]
    type = DirichletBC
    variable = 'disp_y'
    boundary = 'top bottom left right'
    value = 0
  [../]
  [./xfix]
    type = DirichletBC
    variable = 'disp_x'
    boundary = 'bottom'
    value = 0
  [../]
