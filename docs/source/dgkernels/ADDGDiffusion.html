ADDGDiffusion

Use DG to compute a diffusion term, and AD is adopted for Jacobian calculation.

    C++ Type:double

    Options:

    Description:epsilon

  • sigmasigma

    C++ Type:double

    Options:

    Description:sigma

    C++ Type:NonlinearVariableName

    Options:

    Description:The name of the variable that this boundary condition applies to
