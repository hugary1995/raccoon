SimpleACInterface

Gradient energy for Allen-Cahn Kernel with constant Mobility and Interfacial parameter

note

This kernel is part of a set of simplified phase field kernels that have been created for educational purposes. These kernels use simplified formulations with limitations that prohibit spatially varying mobilities and gradient energies. The source code of these kernels is kept simple without using advanced C++ features.

This kernel implements the weak form for the gradient contribution in the Allen-Cahn equation. For a more feature complete version see the ACInterface kernel, which allows for variable dependent mobilities and gradient energy parameters .
