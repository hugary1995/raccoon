FESpaces

FESpaces syntax is used to create MFEM finite element spaces. Valid finite element spaces are H1 for continuous finite elements, L2 for discontinuous finite elements, RT for Raviart-Thomas H(div) conforming finite elements, and ND for Nedelec H(curl) conforming finite elements.
