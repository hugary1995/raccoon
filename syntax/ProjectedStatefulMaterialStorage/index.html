ProjectedStatefulMaterialStorage System

The ProjectedStatefulMaterialStorage Action sets up the required objects to store old material state projected onto finite element basis functions. See the documentation for ProjectedStatefulMaterialStorageAction for details.

Available Actions

  • Moose App
  • ProjectedStatefulMaterialStorageActionMark material properties for projected stateful storage.
