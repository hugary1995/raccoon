Correctors

The Correctors system is designed to modify the values of nonlinear variable solutions. This can be as part of a predictor-corrector time integration scheme for example.

commentnote

Correctors are UserObjects behind the scene. They simply have a dedicated role and syntax.

commentnote

Please let us know on GitHub Discussions how you are using the Correctors system so we can include your techniques on this page!

Available Objects

  • Moose App
  • PointwiseRenormalizeVectorPointwise renormalize the solution of a set of variables comprising a vector

Available Actions

  • Moose App
  • AddCorrectorActionAdd a Corrector object to the simulation.
