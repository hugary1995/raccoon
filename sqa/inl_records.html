INL Software Quality Records

Several software quality-related documents and records are only available with access to the INL network. Instructions on how to access these documents are provided below. These are stored in the same way for each MOOSE-based application, so where appropriate, use the name of the actual application of interest when searching.

warningwarning:INL Network Required

To access the following documents you must be on the INL network.

Software Quality Assurance Plan (SQAP)

The SQAP used is titled the "Software Quality Assurance Plan for MOOSE and MOOSE-Based Applications" (PLN-4005). This plan lays out the specifics of how MOOSE and MOOSE-based applications ensure that life-cycle activities are planned and performed in a traceable and orderly manner. It includes the requirements for Configuration Management Plan (CMP), Verification and Validation Plan (VVP), and Software Quality Plan (SQP). PLN-4005 can be accessed from the INL network as follows.

  1. From the Nucleus Home page select A-Z.

  2. From the alphabetical list select EDMS.

  3. Enter PLN-4005 in the "Document Id" field and select Submit.

  4. To download, select the Doc Viewer down arrow and then select the Record File pdf.

Enterprise Architecture (EA) Record

The EA record can be accessed from the INL network by U.S. citizen employees as follows.

  1. From the Nucleus Home page select A-Z.

  2. From the alphabetical list select Enterprise Architecture (EA) Repository.

  3. Click All to enable a search of all entries.

  4. In the search area directly above All, select Name in the dropdown menu beside the search box.

  5. Enter the name of the application in the search box and press Enter/Return on the keyboard.

  6. When the search is complete, click on the name of the application.

commentnote:Access for foreign national employees

Foreign national INL employees do not have direct access to the EA repository in the same way as U.S. citizen employees. In order to access this system, please perform the following steps.

  1. From the Nucleus Home page select A-Z.

  2. From the alphabetical list select ServiceNow.

  3. Click the Filter/Funnel icon near the top left corner of the page, if the left panel is not expanded.

  4. In the left panel, select Application Portfolio and then All Business Applications

  5. Repeat the search steps in the instructions for U.S. citizens above.

Safety Software Determination (SSD)

The SSD can be accessed from the INL network as follows.

  1. Under the Related Links section on the EA application page, (refer to EA Record), select the Application Documents tab

  2. The Safety Software Determination link will either be listed directly or shown under Other

Quality Level Determination (QLD)

The QLD can be accessed from the INL network as follows.

  1. Under the Related Links section on the EA application page, (refer to EA Record), select the Application Documents tab

  2. The Quality Level Determination link will either be listed directly or shown under Other

Asset Management Plan (AMP)

In accordance with INL Laboratory Wide Procedure (LWP) "Managing Information Technology Assets" (LWP-13620 rev. 18), an Information Technology (IT) asset maintenance plan, or equivalent, must be created by the end user to specify how the software application will be managed after it is deployed for use, see the INL template, "IT Asset Maintenance Plan Template" (TEM-215) for further information.
