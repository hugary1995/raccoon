MOOSEVariableToNEML2

This object requires the application to be built with NEML2.

Description

This object collects a given "moose_variable" for use as an input "neml2_variable" to a NEML2 model executed by a ExecuteNEML2Model user object.

To use a MOOSE material property as input to a NEML2 object see MOOSEMaterialPropertyToNEML2.

Example Input Syntax

[UserObjects]
  [gather_a]
    type = MOOSEVariableToNEML2
    moose_variable = a
    neml2_variable = forces/A
    execute_on = 'INITIAL TIMESTEP_BEGIN'
  []
[]
(moose/modules/solid_mechanics/test/tests/neml2/neml2_to_moose_material.i)
