NEML2Action

To use this object, you need to have the NEML2 library installed. Refer to the documentation for guidance on how to enable it. To build this library MOOSE must be configured with LIBTORCH support! (Original description: Parse and set up NEML2 objects)

Description

NEML2Action is a MOOSE action that constructs objects needed for MOOSE-NEML2 integration. This is invoked by including the NEML2 block at the top level in a MOOSE input file. See the page documenting the syntax for that block for a description, example usage, and parameters.
