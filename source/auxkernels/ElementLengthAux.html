ElementLengthAux

Compute the element size using Elem::hmin() or Elem::hmax() from libMesh.

Description

The element "length" is often needed for creating stabilization coefficients or when adapting the mesh. This will compute the minimum or maximum element length and populate an AuxVariable with the result. The element size calculation uses the libMesh hmin() or hmax() method from the Elem class to compute the length.

Example Syntax

[AuxKernels<<<{"href": "../../syntax/AuxKernels/index.html"}>>>]
  [./min]
    type = ElementLengthAux<<<{"description": "Compute the element size using Elem::hmin() or Elem::hmax() from libMesh.", "href": "ElementLengthAux.html"}>>>
    variable<<<{"description": "The name of the variable that this object applies to"}>>> = min
    method<<<{"description": "The size calculation to perform ('min' or 'max')."}>>> = min
    execute_on<<<{"description": "The list of flag(s) indicating when this object should be executed. For a description of each flag, see https://mooseframework.inl.gov/source/interfaces/SetupInterface.html."}>>> = initial
  [../]
  [./max]
    type = ElementLengthAux<<<{"description": "Compute the element size using Elem::hmin() or Elem::hmax() from libMesh.", "href": "ElementLengthAux.html"}>>>
    variable<<<{"description": "The name of the variable that this object applies to"}>>> = max
    method<<<{"description": "The size calculation to perform ('min' or 'max')."}>>> = max
    execute_on<<<{"description": "The list of flag(s) indicating when this object should be executed. For a description of each flag, see https://mooseframework.inl.gov/source/interfaces/SetupInterface.html."}>>> = initial
  [../]
[../]
