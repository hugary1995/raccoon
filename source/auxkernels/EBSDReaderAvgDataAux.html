EBSDReaderAvgDataAux

buildconstruction:Undocumented Class

The EBSDReaderAvgDataAux has not been documented. The content listed below should be used as a starting point for documenting the class, which includes the typical automatic documentation associated with a MooseObject; however, what is contained is ultimately determined by what is necessary to make the documentation clear for users.

Outputs the requested EBSD reader average data, for a given phase if specified, for the grain at the local node/element.
