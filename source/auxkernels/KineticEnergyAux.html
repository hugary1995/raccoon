KineticEnergyAux

Compute the kinetic energy of continuum-based finite elements

Description

This object is used to compute the value of the kinetic energy of continuum finite elements using elemental variables. This auxiliary kernel can be used in conjunction with ElementIntegralVariablePostprocessor in order to compute a single kinetic energy value corresponding to a block or various blocks.

Input Parameters

  • newmark_velocity_xX component of the velocity from the Newmark integration scheme

    C++ Type:std::vector<VariableName>

    Unit:(no unit assumed)

    Controllable:No

    Description:X component of the velocity from the Newmark integration scheme

  • newmark_velocity_yY component of the velocity from the Newmark integration scheme

    C++ Type:std::vector<VariableName>

    Unit:(no unit assumed)

    Controllable:No

    Description:Y component of the velocity from the Newmark integration scheme

  • newmark_velocity_zZ component of the velocity from the Newmark integration scheme

    C++ Type:std::vector<VariableName>

    Unit:(no unit assumed)

    Controllable:No

    Description:Z component of the velocity from the Newmark integration scheme

  • variableThe name of the variable that this object applies to

    C++ Type:AuxVariableName

    Unit:(no unit assumed)

    Controllable:No

    Description:The name of the variable that this object applies to

  • check_boundary_restrictedTrueWhether to check for multiple element sides on the boundary in the case of a boundary restricted, element aux variable. Setting this to false will allow contribution to a single element's elemental value(s) from multiple boundary sides on the same element (example: when the restricted boundary exists on two or more sides of an element, such as at a corner of a mesh

    Default:True

    C++ Type:bool

    Controllable:No

    Description:Whether to check for multiple element sides on the boundary in the case of a boundary restricted, element aux variable.     Default:density

    Unit:(no unit assumed)

    Controllable:No

    Description:Name of material property or a constant real number defining the density of the continuum material.

    Controllable:No

    Description:The list of flag(s) indicating when this object should be executed. For a description of each flag, see https://mooseframework.inl.gov/source/interfaces/SetupInterface.html.
