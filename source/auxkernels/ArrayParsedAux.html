ArrayParsedAux

Sets field array variable values to the evaluation of a parsed expression.

Overview

This auxiliary kernel is meant to emulate the behavior and syntax of ParsedAux for array variables. Using "expression", one can define a general parsed function dependent on array variables, scalar field variables, mesh coordinates, and time:

where is the array variable specified in "variable" with components, is the component index of the variable, are coupled array variables specified in "coupled_array_variables", are coupled field variables specified in "coupled_variables", and , , , and are coordinates used if "use_xyzt" is true.

Example Input File Syntax

This is an example that defines the following expression:

[AuxKernels<<<{"href": "../../syntax/AuxKernels/index.html"}>>>]
  [parsed_aux]
    type = ArrayParsedAux<<<{"description": "Sets field array variable values to the evaluation of a parsed expression.", "href": "ArrayParsedAux.html"}>>>
    variable<<<{"description": "The name of the variable that this object applies to"}>>> = parsed
    expression<<<{"description": "Parsed function expression to compute"}>>> = '(u^2 + v)*(x - const)*factor'
    coupled_array_variables<<<{"description": "Vector of coupled array variable names."}>>> = 'u v'
    coupled_variables<<<{"description": "Vector of coupled variable names."}>>> = const
    constant_names<<<{"description": "Vector of constants used in the parsed function (use this for kB etc.)"}>>> = 'factor'
    constant_expressions<<<{"description": "Vector of values for the constants in constant_names (can be an FParser expression)"}>>> = '3.14'
    use_xyzt<<<{"description": "Make coordinate (x,y,z) and time (t) variables available in the function expression."}>>> = true
  []
  [sum_aux]
    type = ArrayVarReductionAux<<<{"description": "Takes an array variable and performs a reduction operation on it (max, min, sum, average) and stores as a standard variable.", "href": "ArrayVarReductionAux.html"}>>>
    variable<<<{"description": "The name of the variable that this object applies to"}>>> = sum
    array_variable<<<{"description": "Array variable to process."}>>> = parsed
  []
[]

[Mesh]
  [gmg]
    type = GeneratedMeshGenerator
    dim = 1
    nx = 10
  []
[]

[Variables]
  [u]
    components = 2
  []
  [v]
    components = 2
  []
[]

[AuxVariables]
  [const]
    initial_condition = 0.5
  []
  [parsed]
    components = 2
  []
  [sum]
  []
[]

[Functions]
  [cosx]
    type = ParsedFunction
    expression = 'cos(x)'
  []
  [sinx]
    type = ParsedFunction
    expression = 'sin(x)'
  []
  [px]
    type = ParsedFunction
    expression = 'x'
  []
  [mx]
    type = ParsedFunction
    expression = '-x'
  []
[]

[ICs]
  [uic]
    type = ArrayFunctionIC
    variable = u
    function = 'cosx sinx'
  []
  [vic]
    type = ArrayFunctionIC
    variable = v
    function = 'px mx'
  []
[]

[AuxKernels]
  [parsed_aux]
    type = ArrayParsedAux
    variable = parsed
    expression = '(u^2 + v)*(x - const)*factor'
    coupled_array_variables = 'u v'
    coupled_variables = const
    constant_names = 'factor'
    constant_expressions = '3.14'
    use_xyzt = true
  []
  [sum_aux]
    type = ArrayVarReductionAux
    variable = sum
    array_variable = parsed
  []
[]

[Postprocessors]
  [avg]
    type = ElementAverageValue
    variable = sum
  []
[]

[Outputs]
  exodus = true
[]

[Problem]
  solve = false
[]

[Executioner]
  type = Steady
