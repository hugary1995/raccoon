MaterialRateRealAux

Outputs element material properties rate of change

Description

The MaterialRateRealAux AuxKernel is used to output the rate of change of material properties as an element-level, variable. The rate is computed by

where is the time step size and "property" is the material property being evaluated. This class derives from the same base class as MaterialRealAux and therefore also has optional parameters "factor" and "offset".

Example Input Syntax

[AuxKernels<<<{"href": "../../syntax/AuxKernels/index.html"}>>>]
  [rate]
    type = MaterialRateRealAux<<<{"description": "Outputs element material properties rate of change", "href": "MaterialRateRealAux.html"}>>>
    variable<<<{"description": "The name of the variable that this object applies to"}>>> = rate
    property<<<{"description": "The material property name."}>>> = prop

    property = prop
  []

  []
[]

    type = ParsedFunction
    expression = t*t/2
  []
[]

[Kernels]

    type = CoefDiffusion
    variable = u
    coef = 0.1
  []
  [time]
    type = TimeDerivative
    variable = u
  []
[]

    variable = u
    boundary = left
    value = 0
  []

    variable = u

    value = 1
  []
[]

    prop_names = prop
    prop_values = func
    block = 0
  []
