ParsedScalarReporter

Applies parsed functions to scalar entries held in reporters.

Overview

ParsedScalarReporter operates on scalars contained from a series of scalar reporters using a parsed function expression as shown in Listing 1. This reporter allows several scalar reporters to be combined into a single scalar. Variables in the parsed function expression are given by the "scalar_reporter_symbols". There is one "scalar_reporter_names" for each "scalar_reporter_symbols". This is a scalar version of the ParsedVectorReporter reporter.

Optimization use case

This reporter was created to compute the objective function from the misfit reporter vector created by OptimizationData. The scalar reporter created can then be transferred as the objective value into GeneralOptimization.
