MeshMetaDataReporter

A Reporter object for outputting mesh metadata in a human-readable form. By default, it executes only on initial.

This object is a specialization of RestartableDataReporter, which you can see for a more detailed parameter description.

See MeshMetaDataInterface for more information on the use of mesh metadata.

Example Input Syntax

The following input file snippet demonstrates the use of the MeshMetaDataReporter object.

[Reporters<<<{"href": "../../syntax/Reporters/index.html"}>>>]
  [metadata]
    type = MeshMetaDataReporter<<<{"description": "Reports the mesh meta data.", "href": "MeshMetaDataReporter.html"}>>>
  []
[]
(moose/test/tests/reporters/mesh_meta_data_reporter/mesh_meta_data_reporter.i)

Input Parameters

  • allow_unimplementedFalseSet to true to allow the empty output of data that does not have a JSON output implementation

    Default:False

    C++ Type:bool

    Controllable:No

    Description:Set to true to allow the empty output of data that does not have a JSON output implementation

    Controllable:No

    Description:The entries to output for each restartable value

    C++ Type:std::vector<std::string>

    Controllable:No

    Description:The data name patterns to exclude (* for matching all, ? for matching a single character)

  • includeThe data name patterns to include (* for matching all, ? for matching a single character)

    C++ Type:std::vector<std::string>

    Controllable:No

    Description:The data name patterns to include (* for matching all, ? for matching a single character)

    Controllable:No

    Description:The list of flag(s) indicating when this object should be executed. For a description of each flag, see https://mooseframework.inl.gov/source/interfaces/SetupInterface.html.
