ElementVariableStatistics

Element reporter to get statistics for a coupled variable. This can be transfered to other apps.

Overview

ElementVariableStatistics produces the following statistics for a variable: Maximum, Minium, Average, Integral, Total Elements. The "base_name" can be used to prepend a name to each reporter.

Example Input File Syntax

[Reporters<<<{"href": "../../syntax/Reporters/index.html"}>>>]
  [Reporters]
    [elem_stats]
      type = ElementVariableStatistics
      coupled_var = u
      base_name = diffusion
    []
  []
[]
(moose/test/tests/reporters/element_reporter/elem_stats.i)

  • coupled_varCoupled variable whose value is used.

    C++ Type:std::vector<VariableName>

    Unit:(no unit assumed)

    Controllable:No

    Description:Coupled variable whose value is used.
