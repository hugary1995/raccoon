ThermalSensitivity

Computes cost sensitivity needed for multimaterial SIMP method.

Overview

TThis material takes a DerivativeParsedMaterial and uses its derivative with respect to the "design_density" variable. The 'design_density' can be a pseudo-density which is employed to solve a solid isotropic material penalization (SIMP) topology optimization problem. This sensitivity is intended to optimize a thermal compliance objective function.

Example Input File Syntax

This material can be used to optimize a multimaterial topology problem or in combination with other physics, such as a small deformation elastic problem

[Materials]
  [tc]
    type = ThermalSensitivity
    design_density = mat_den
    thermal_conductivity = thermal_cond
    temperature = temp
    outputs = 'exodus'
  []
[]
(moose/modules/combined/test/tests/optimization/compliance_sensitivity/three_materials_thermal.i)

[Materials]
  [thermal_cond]
    type = DerivativeParsedMaterial
    # ordered multimaterial simp
