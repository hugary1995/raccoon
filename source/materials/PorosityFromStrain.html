PorosityFromStrain

Porosity calculation from the inelastic strain.

Description

PorosityFromStrain computes the porosity, , from the combined inelastic strain, :

Here, is typically computed by ComputeMultipleInelasticStress.

Example Input Files

[Materials]
  [porosity]
    type = PorosityFromStrain
    initial_porosity = 0
    inelastic_strain = strain
    outputs = all
  []
  [strain]
    type = GenericFunctionRankTwoTensor
    tensor_name = strain
    tensor_functions = 'volumetric'
    outputs = all
  []
[]
