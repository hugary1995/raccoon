ADNonlocal Damage

Nonlocal damage model. Given an RadialAverage UO this creates a new damage index that can be used as for ComputeDamageStress without havign to change existing local damage models.

Description

ADNonlocalDamage is a model to define the effect of damage on the stress and stiffness in a continuum damage mechanics setting. Uses automatic differentiation material properties. Similar to the NonlocalDamage object except that the material properties generated can be used with automatic differentiation.

  • average_UORadial Average user object

    C++ Type:UserObjectName

    Controllable:No

    Description:Radial Average user object

  • local_damage_modelName of the local damage model used to compute the nonlocal damage index

    C++ Type:MaterialName

    Controllable:No

    Description:Name of the local damage model used to compute the nonlocal damage index
