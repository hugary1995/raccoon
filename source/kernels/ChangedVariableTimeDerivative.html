ChangedVariableTimeDerivative

A modified time derivative Kernel that multiplies the time derivative bythe derivative of the nonlinear preconditioning function

Overview

This kernel implements the time derivative of a variable that has been converted using a change of variable transformation. So, we transform variable to . It is used in kinetic equations that use a change of variable.

The time derivative becomes The derivative of the transformation is accessed from a material property, like the linearized interface function implemented in LinearizedInterfaceFunction

Example Input File Syntax

The kernel is defined in the input file using the syntax:

[Kernels<<<{"href": "../../syntax/Kernels/index.html"}>>>]
  [phi0_ACInt]
    type = ACInterfaceChangedVariable<<<{"description": "Gradient energy Allen-Cahn Kernel using a change of variable", "href": "ACInterfaceChangedVariable.html"}>>>
    variable<<<{"description": "The name of the variable that this residual object operates on"}>>> = phi0
    kappa_name<<<{"description": "The kappa used with the kernel"}>>> = kappa_op
    mob_name<<<{"description": "The mobility used with the kernel"}>>> = L
    order_parameter<<<{"description": "Order parameter material defnining the change of variable function"}>>> = gr0
  []
[]
(moose/modules/phase_field/test/tests/grain_growth_w_linearized_interface/grain_growth_linearized_interface.i)

Alternatively, this kernel is added automatically by the GrainGrowthLinearizedInterfaceAction.

This is fast to determine via a pregenerated node-to-elem map and is robust on conforming meshes. In the optional all_proximate_sides algorithm, primary boundary elements are searched iff they touch that nearest node, even if they are not topologically connected to it.
