Out-Of-Plane Pressure

Apply pressure in the out-of-plane direction in 2D plane stress or generalized plane strain models

Description

The OutOfPlanePressure kernel applies an out-of-plane pressure value in the out-of-plane direction of Generalized Plane Strain or 2D plane stress problems. Either a function or a postprocessor can be used to specify the value of the out-of-plane pressure at each quadrature point. (1) where is the computed pressure, is the function or postprocessor value of the pressure to be applied, and is the unit normal vector to the out-of-plane surface. Following the convention of the Pressure boundary condition, the unit normal vector is considered to be positive when pointing inward towards the surface.

Input Parameters

  • variableThe name of the variable that this residual object operates on

    C++ Type:NonlinearVariableName

    Unit:(no unit assumed)

    Controllable:No

    Description:The name of the variable that this residual object operates on

    Unit:(no unit assumed)

    Controllable:No

    Description:The displacements

    Default:1

    C++ Type:double

    Unit:(no unit assumed)

    Controllable:No

    Default:1.0

    C++ Type:FunctionName

    Unit:(no unit assumed)

    Controllable:No

    Description:Function used to prescribe pressure

  • matrix_onlyFalseWhether this object is only doing assembly to matrices (no vectors)

    Default:False

    C++ Type:bool

    Controllable:No

    Description:Whether this object is only doing assembly to matrices (no vectors)

  • postprocessorPostprocessor used to prescribe pressure

    C++ Type:PostprocessorName

    Unit:(no unit assumed)

    Controllable:No

    Description:Postprocessor used to prescribe pressure
