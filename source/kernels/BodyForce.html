Example Syntax

The case below demonstrates the use of BodyForce where the force term is supplied based upon a function form:

[Kernels]
  [./diff]
    type = Diffusion
    variable = u
  [../]

  [./body_force]
    type = BodyForce
    variable = u
    block = 1
    value = 10
    function = 'x+y'
  [../]

  [./time]
    type = TimeDerivative
    variable = u
  [../]
