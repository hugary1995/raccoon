MatrixSymmetryCheck

This class reports whether the system matrix is symmetric or not. If it is symmetric then the value returned is 1, else it is 0. This class's execution is fairly expensive so we only recommend using it on relatively small problems.

Input Parameters

  • matThe petsc binary mat file containing the matrix. If this parameter is not provided, then the system matrix is used

    C++ Type:std::string

    Controllable:No

    Description:The petsc binary mat file containing the matrix. If this parameter is not provided, then the system matrix is used

  • mat_number_to_load1A binary file may contain multiple writes of a matrix. This parameter can be used to load a particular matrix from the binary file. By default we load the first written matrix

    Default:1

    C++ Type:unsigned int

    Controllable:No

    Description:A binary file may contain multiple writes of a matrix. This parameter can be used to load a particular matrix from the binary file. By default we load the first written matrix

  • symmetry_tol1e-08The tolerance (both relative and absolute) for comparing symmetry

    Default:1e-08

    C++ Type:double

    Unit:(no unit assumed)

    Controllable:No

    Description:The tolerance (both relative and absolute) for comparing symmetry
