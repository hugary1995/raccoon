NodalVariableValue

Outputs values of a nodal variable at a particular location

Description

In order to obtain the value of a nodal variable at a particular location (i.e., temperature and displacement) a NodalVariableValue postprocessor is used. For example, this postprocessor is useful for obtaining the centerline temperature at the location of a thermocouple to compare against experimental data.

Example Input Syntax

[Postprocessors<<<{"href": "../../syntax/Postprocessors/index.html"}>>>]
  [./off_block]
    type = NodalVariableValue<<<{"description": "Outputs values of a nodal variable at a particular location", "href": "NodalVariableValue.html"}>>>
    variable<<<{"description": "The variable to be monitored"}>>> = v
    nodeid<<<{"description": "The ID of the node where we monitor"}>>> = 0
  [../]
[]
(moose/test/tests/misc/check_error/nodal_value_off_block.i)
