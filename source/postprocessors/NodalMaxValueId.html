NodalMaxValueId

Finds the node id with the maximum nodal value across all postprocessors.

This postprocessor performs a reduction across all ranks to compute the maximum, then returns the id of the node with the global maximum.

Example input syntax

In this input file we obtain both the maximum value of variable u using a NodalExtremeValue and the location where it is reached using a NodalMaxValueId.

[Postprocessors<<<{"href": "../../syntax/Postprocessors/index.html"}>>>]
  [./max_nodal_pps]
    type = NodalExtremeValue<<<{"description": "Finds either the min or max elemental value of a variable over the domain.", "href": "NodalExtremeValue.html"}>>>
    variable<<<{"description": "The name of the variable that this postprocessor operates on"}>>> = u
  [../]
  [./max_node_id]
    type = NodalMaxValueId<<<{"description": "Finds the node id with the maximum nodal value across all postprocessors.", "href": "NodalMaxValueId.html"}>>>
    variable<<<{"description": "The name of the variable that this postprocessor operates on"}>>> = u
  [../]
  [./min_nodal_pps]
    type = NodalExtremeValue<<<{"description": "Finds either the min or max elemental value of a variable over the domain.", "href": "NodalExtremeValue.html"}>>>
    variable<<<{"description": "The name of the variable that this postprocessor operates on"}>>> = u
    value_type<<<{"description": "Type of extreme value to return. 'max' returns the maximum value. 'min' returns the minimum value. 'max_abs' returns the maximum of the absolute value."}>>> = min
  [../]
[]
(moose/test/tests/postprocessors/nodal_extreme_value/nodal_extreme_pps_test.i)
