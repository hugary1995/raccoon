GreaterThanLessThanPostprocessor

Count number of DOFs of a non-linear variable that are greater than or less than a given threshold

Description

The GreaterThanLessThanPostprocessor computes the total number of degrees of freedom of a nonlinear variable that are greater than or less than the parameter value. The comparison of greater or less is specified using the MooseEnum parameter comparator; the possible values of comparator are greater or less. The computation may also be restricted to a subdomain using the parameter subdomain.

Example Syntax

[Postprocessors<<<{"href": "../../syntax/Postprocessors/index.html"}>>>]
  [active_lm]
    type = GreaterThanLessThanPostprocessor<<<{"description": "Count number of DOFs of a non-linear variable that are greater than or less than a given threshold", "href": "GreaterThanLessThanPostprocessor.html"}>>>
    variable<<<{"description": "The name of the variable to conduct a comparison for"}>>> = lm
    execute_on<<<{"description": "The list of flag(s) indicating when this object should be executed. For a description of each flag, see https://mooseframework.inl.gov/source/interfaces/SetupInterface.html."}>>> = 'nonlinear timestep_end'
    value<<<{"description": "The value to compare against"}>>> = 1e-8
  []
  [violations]
    type = GreaterThanLessThanPostprocessor<<<{"description": "Count number of DOFs of a non-linear variable that are greater than or less than a given threshold", "href": "GreaterThanLessThanPostprocessor.html"}>>>
    variable<<<{"description": "The name of the variable to conduct a comparison for"}>>> = u
    execute_on<<<{"description": "The list of flag(s) indicating when this object should be executed. For a description of each flag, see https://mooseframework.inl.gov/source/interfaces/SetupInterface.html."}>>> = 'nonlinear timestep_end'
    value<<<{"description": "The value to compare against"}>>> = -1e-8
    comparator<<<{"description": "The comparison to perform between the variable and the provided value"}>>> = 'less'
  []
[]
