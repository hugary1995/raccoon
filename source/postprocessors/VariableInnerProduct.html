VariableInnerProduct

Computes a volume integral of the specified variable

The inner product computed is for the L2 norm:

where and are the variable and second_variable parameters and is the integration domain, either the whole domain or the specified blocks.

Example input syntax

In this input file, we compute the inner products and of auxiliary variables. We verify that is equal to the L2 norm.

[Postprocessors<<<{"href": "../../syntax/Postprocessors/index.html"}>>>]
  [./f_dot_g]
    type = VariableInnerProduct<<<{"description": "Computes a volume integral of the specified variable", "href": "VariableInnerProduct.html"}>>>
    variable<<<{"description": "The name of the variable that this object operates on"}>>> = f
    second_variable<<<{"description": "The name of the second variable in the inner product (variable, second_variable)"}>>> = g
  [../]

  [./f_dot_f]
    type = VariableInnerProduct<<<{"description": "Computes a volume integral of the specified variable", "href": "VariableInnerProduct.html"}>>>
    variable<<<{"description": "The name of the variable that this object operates on"}>>> = f
    second_variable<<<{"description": "The name of the second variable in the inner product (variable, second_variable)"}>>> = f
  [../]

  [./norm_f]
    type = ElementL2Norm<<<{"description": "Computes a volume integral of the specified variable", "href": "ElementL2Norm.html"}>>>
    variable<<<{"description": "The name of the variable that this object operates on"}>>> = f
  [../]
