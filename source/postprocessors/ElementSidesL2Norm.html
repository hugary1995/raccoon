Computes the L2 norm of a variable over element sides.

These terms are initialized on each side, so the term accounts for the geometry/area of the element, the for the weight in the quadrature, for any eventual coordinate change (cylindrical, spherical) and is the value of the variable at the quadrature point.

Example input syntax

In this example, we compute the L2 norm for the Lagrange multiplier variable using an ElementSideL2Norm postprocessor.

[Postprocessors]
  [intu]
    type = ElementIntegralVariablePostprocessor
    variable = u
    block = 0
  []
  [lambdanorm]
    type = ElementSidesL2Norm
    variable = lambda
  []
[]
