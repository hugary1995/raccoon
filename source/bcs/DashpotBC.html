Model a dashpot boundary condition where the traction is proportional to the normal velocity.

where is the friction coefficient, the local surface normal and is the velocity, computed from the displacements.

commentnote

This boundary condition hard-codes an implicit Euler time integration scheme in its contribution to the Jacobian.

Input Parameters

  • boundaryThe list of boundary IDs from the mesh where this object applies

    C++ Type:std::vector<BoundaryName>

    Controllable:No

    Description:The list of boundary IDs from the mesh where this object applies

  • componentThe displacement component corresponding the variable this BC acts on.

    C++ Type:unsigned int

    Controllable:No

    Description:The displacement component corresponding the variable this BC acts on.

  • disp_xDisplacement in the x direction

    C++ Type:std::vector<VariableName>

    Unit:(no unit assumed)

    Controllable:No

    Description:Displacement in the x direction

  • variableThe name of the variable that this residual object operates on

    C++ Type:NonlinearVariableName

    Unit:(no unit assumed)

    Controllable:No

    Description:The name of the variable that this residual object operates on

  • coefficient1The viscosity coefficient

    Default:1

    C++ Type:double

    Unit:(no unit assumed)

    Controllable:No

    Description:The viscosity coefficient

  • disp_yDisplacement in the y direction

    C++ Type:std::vector<VariableName>

    Unit:(no unit assumed)

    Controllable:No

    Description:Displacement in the y direction

  • disp_zDisplacement in the z direction

    C++ Type:std::vector<VariableName>

    Unit:(no unit assumed)

    Controllable:No

    Description:Displacement in the z direction

  • displacementsThe displacements

    C++ Type:std::vector<VariableName>

    Unit:(no unit assumed)

    Controllable:No

    Description:The displacements

    Default:False

    C++ Type:bool

    Controllable:No

    Description:Whether this object is only doing assembly to matrices (no vectors)
