CoupledVarNeumannBC

Imposes the integrated boundary condition , where is a variable.

Description

CoupledVarNeumannBC is a generalization of the NeumannBC. Instead of coupling a single real value through the value parameter, a coupled variable is provided for the v parameter.

Parameter can be used to scale the boundary condition value. By default, it is (i.e. no scaling). Note that is actually a field variable, so spatially dependent scaling is possible. This can be used to locally turn the BC on or off.

Example Input Syntax

  [./right]
    type = CoupledVarNeumannBC
    variable = u
    boundary = 1
    v = coupled_bc_var
  [../]
(moose/test/tests/bcs/coupled_var_neumann/coupled_var_neumann.i)

  ymax = 1

  [../]
[]

    variable = u
  [../]
[]

  [./coupled_bc_var]
  [../]
[]

[ICs]
  [./coupled_bc_var]
    type = FunctionIC
    variable = coupled_bc_var
    function = set_coupled_bc_var
  [../]
[]

    type = ParsedFunction
    expression = 'y - 0.5'
  [../]

    variable = u
    boundary = 3
    value = 0
  [../]

    boundary = 1
