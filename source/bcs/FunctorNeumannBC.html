FunctorNeumannBC

Imposes the integrated boundary condition , where is a functor.

Description

FunctorNeumannBC is a generalization of ADNeumannBC which is used for imposing flux boundary conditions on systems of partial differential equations (PDEs) where the flux is represented by a spatially- and temporally-varying MOOSE Function. That is, for a PDE of the form where is the domain, and is its boundary, a FunctorNeumannBC object can be used to impose the third condition if the function is well-defined for all relevant times and . In this case, the parameters "functor" and "coefficient" are functors, whose product represents the mathematical function (or if "flux_is_inward" is set to false), and the user must define one or more sidesets corresponding to the boundary subset via the "boundary" parameter.

Input Parameters

  • boundaryThe list of boundary IDs from the mesh where this object applies

    C++ Type:std::vector<BoundaryName>

    Controllable:No

    Description:The list of boundary IDs from the mesh where this object applies

  • functorThe functor to impose. A functor is any of the following: a variable, a functor material property, a function, a post-processor, or a number.

    C++ Type:MooseFunctorName

    Unit:(no unit assumed)

    Controllable:No

    Description:The functor to impose. A functor is any of the following: a variable, a functor material property, a function, a post-processor, or a number.

  • variableThe name of the variable that this residual object operates on

    C++ Type:NonlinearVariableName

    Unit:(no unit assumed)

    Controllable:No

    Description:The name of the variable that this residual object operates on
