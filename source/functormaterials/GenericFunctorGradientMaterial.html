GenericFunctorGradientMaterial

FunctorMaterial object for declaring properties that are populated by evaluation of gradients of Functors (a constant, variable, function or functor material property) objects.

Overview

This object creates vector functor material properties that are gradients of other functors.

commentnote

All AD-types of the properties defined in this material must match. Variables are automatically considered as AD functors, even auxiliary variables. The AD version of this material is ADGenericFunctorGradientMaterial. Its inputs are a vector of AD functors and it creates AD vector functor material properties.

Example Input File Syntax

In this example, ADGenericFunctorGradientMaterial is used to compute the gradient of the variable u, which is then multiplied by a diffusion coefficient to obtain the diffusive flux. This flux is computed at the element centroids, not one the element faces.

[Materials<<<{"href": "../../syntax/Materials/index.html"}>>>]
  [functor_properties]
    type = ADGenericFunctorMaterial<<<{"description": "FunctorMaterial object for declaring properties that are populated by evaluation of a Functor (a constant, variable, function or functor material property) objects.", "href": "GenericFunctorMaterial.html"}>>>
    prop_names<<<{"description": "The names of the properties this material will have"}>>> = 'sink_mat diffusive_coef'
    prop_values<<<{"description": "The corresponding names of the functors that are going to provide the values for the variables"}>>> = 'sink 4.5'
  []
  [gradient_of_u]
    type = ADGenericFunctorGradientMaterial<<<{"description": "FunctorMaterial object for declaring properties that are populated by evaluation of gradients of Functors (a constant, variable, function or functor material property) objects.", "href": "GenericFunctorGradientMaterial.html"}>>>
    prop_names<<<{"description": "The names of the properties this material will have"}>>> = 'grad_u'
    prop_values<<<{"description": "The corresponding names of the functors which gradient are going to provide the values for the variables"}>>> = 'u'
  []
[]

[AuxKernels<<<{"href": "../../syntax/AuxKernels/index.html"}>>>]
  [diffusive_flux_x]
    type = ADFunctorVectorElementalAux<<<{"description": "Evaluates a vector functor (material property usually) on the current element.For finite volume, this evaluates the vector functor at the centroid.", "href": "../auxkernels/FunctorVectorElementalAux.html"}>>>
    variable<<<{"description": "The name of the variable that this object applies to"}>>> = 'diffusive_flux_x'
    functor<<<{"description": "The functor to evaluate. A functor is any of the following: a variable, a functor material property, a function, a post-processor, or a number."}>>> = 'grad_u'
    factor<<<{"description": "A factor to apply on the functor. A functor is any of the following: a variable, a functor material property, a function, a post-processor, or a number."}>>> = 'diffusive_coef'
    component<<<{"description": "Component of the vector functor"}>>> = 0
  []
  [diffusive_flux_y]
    type = ADFunctorVectorElementalAux<<<{"description": "Evaluates a vector functor (material property usually) on the current element.For finite volume, this evaluates the vector functor at the centroid.", "href": "../auxkernels/FunctorVectorElementalAux.html"}>>>
    variable<<<{"description": "The name of the variable that this object applies to"}>>> = 'diffusive_flux_y'
    functor<<<{"description": "The functor to evaluate. A functor is any of the following: a variable, a functor material property, a function, a post-processor, or a number."}>>> = 'grad_u'
    factor<<<{"description": "A factor to apply on the functor. A functor is any of the following: a variable, a functor material property, a function, a post-processor, or a number."}>>> = 'diffusive_coef'
    component<<<{"description": "Component of the vector functor"}>>> = 1
  []
  [diffusive_flux_magnitude]
    type = VectorMagnitudeAux<<<{"description": "Creates a field representing the magnitude of three coupled variables using an Euclidean norm.", "href": "../auxkernels/VectorMagnitudeAux.html"}>>>
    variable<<<{"description": "The name of the variable that this object applies to"}>>> = 'diffusive_flux_magnitude'
    x<<<{"description": "x-component of the vector"}>>> = 'diffusive_flux_x'
    y<<<{"description": "y-component of the vector"}>>> = 'diffusive_flux_y'
  []
