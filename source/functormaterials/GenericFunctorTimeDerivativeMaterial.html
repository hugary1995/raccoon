GenericFunctorTimeDerivativeMaterial

FunctorMaterial object for declaring properties that are populated by evaluation of time derivatives of Functors objects. (such as variables, constants, postprocessors). The time derivative is only returned if the 'dot' functor routine is implemented.

Overview

This object creates functor material properties that are time derivatives of other functors.

warningwarning

Not all functors can be used as inputs. Many functors, notably off-the-shelf postprocessors do not compute their time derivative by default. These time derivatives can be derived and implemented as the dot functor routine. Unless this routine is implemented, these functors should not be used as inputs to this functor material.

warningwarning

Time derivatives are not available at all times during the simulation. Notably, on INITIAL and TIMESTEP_BEGIN, the time derivatives are a priori NOT available, and any functor material properties created by this object will return 0.

warningwarning

The time derivative routine used by this functor, namely functor.dot(spatial_argument, state_argument), is not implemented for all spatial arguments and all state arguments for every single functor. Users must be extremely careful in their use of the functors created by this functor material.

commentnote

All AD-types of the properties defined in this material must match. Variables are automatically considered as AD functors, even auxiliary variables. The AD version of this material is ADGenericFunctorTimeDerivativeMaterial. Its inputs are a vector of AD functors and it creates AD functor material properties.

    Controllable:No

    Description:The list of flag(s) indicating when this object should be executed. For a description of each flag, see https://mooseframework.inl.gov/source/interfaces/SetupInterface.html.

    Controllable:No

    Description:The names of the properties this material will have

    Controllable:No
