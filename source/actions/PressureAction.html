Pressure Action

Description

The Pressure Action is used to create a set of pressure boundary conditions for a string of displacement variables; the typical use case for this action is the application of hydrostatic pressure. See the description, example use, and parameters on the Pressure action system page.

warningwarning

When using the Controls system to control the active status of Pressure boundary conditions, the Pressure boundary conditions created by the Pressure Action cannot be controlled individually, they will all follow the same enabled/disabled status.
