Cohesive Zone Action

Description

The CohesiveZoneAction simplifies the input file syntax by adding the appropriate CZMInterfaceKernels, ComputeDisplacementJump, and ComputeGlobalTraction objects depending on the selected kinematic. See CohesiveZone for more details.

Action to create an instance of the cohesive zone model kernel for each displacement component
