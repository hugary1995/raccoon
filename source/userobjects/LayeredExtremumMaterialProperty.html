LayeredExtremumMaterialProperty

Compute material property extrema over layers.

How to define the layers

See the LayeredAverage documentation.

How to retrieve the result

The result of a LayeredExtremumMaterialProperty computation can be saved in an auxiliary variable using a SpatialUserObjectAux. It can be output to a CSV file using a SpatialUserObjectVectorPostprocessor.

Additional computation options

See the LayeredAverage documentation for more sampling option. The averaging operation is replaced by an extremum (min or max) operation.

Example input syntax

In this example, the minimum of material property mat is taken over the whole domain in direction y over ten layers. The result of this averaging in stored in the variable layered_extremum using a SpatialUserObjectAux, and output to a CSV file using a SpatialUserObjectVectorPostprocessor

[UserObjects<<<{"href": "../../syntax/UserObjects/index.html"}>>>]
  [layered_uo]
    type = LayeredExtremumMaterialProperty<<<{"description": "Compute material property extrema over layers.", "href": "LayeredExtremumMaterialProperty.html"}>>>
    direction<<<{"description": "The direction of the layers."}>>> = y
    num_layers<<<{"description": "The number of layers."}>>> = 10
    mat_prop<<<{"description": "Material property for which to find the extreme"}>>> = mat
    value_type<<<{"description": "Type of extreme value to return: 'max' returns the maximum value and 'min' returns the minimum value."}>>> = 'min'
    execute_on<<<{"description": "The list of flag(s) indicating when this object should be executed. For a description of each flag, see https://mooseframework.inl.gov/source/interfaces/SetupInterface.html."}>>> = 'INITIAL LINEAR'
  []
[]

[AuxKernels<<<{"href": "../../syntax/AuxKernels/index.html"}>>>]
  [liaux]
    type = SpatialUserObjectAux<<<{"description": "Populates an auxiliary variable with a spatial value returned from a UserObject spatialValue method.", "href": "../auxkernels/SpatialUserObjectAux.html"}>>>
    variable<<<{"description": "The name of the variable that this object applies to"}>>> = layered_extremum
    user_object<<<{"description": "The UserObject UserObject to get values from.  Note that the UserObject _must_ implement the spatialValue() virtual function!"}>>> = layered_uo
    execute_on<<<{"description": "The list of flag(s) indicating when this object should be executed. For a description of each flag, see https://mooseframework.inl.gov/source/interfaces/SetupInterface.html."}>>> = 'INITIAL LINEAR'
  []
[]

[VectorPostprocessors<<<{"href": "../../syntax/VectorPostprocessors/index.html"}>>>]
  [output]
    type = SpatialUserObjectVectorPostprocessor<<<{"description": "Outputs the values of a spatial user object in the order of the specified spatial points", "href": "../vectorpostprocessors/SpatialUserObjectVectorPostprocessor.html"}>>>
    userobject<<<{"description": "The userobject whose values are to be reported"}>>> = layered_uo
  []
[]
(moose/test/tests/userobjects/layered_extremum/layered_extremum_matprop.i)
