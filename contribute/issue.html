Bug Report / Feature Request

Contribution isn't only about writing code. If you identify a bug, or feel an important feature is missing, tell us and it is a great way to contribute. Go to RACCOON issue page for a list of currently open and closed issues.

Click on the New issue button to submit a ticket. Note that two templates are available, one for Bug Report and the other one for Feature Request.
